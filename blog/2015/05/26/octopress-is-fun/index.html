Octopress Is Fun!

Well, it’s really new way to maintain your blog. Forget about wordpress or any another and mind-broken overweight piece of PHP code, self-hosted blogs, site maintaining, lost database passwords, extremely high traffic from page with your kitty pic and alarms from provider about “extending your plan immediately!” and so on. You just don’t need it anymore.

Github account, some basic ruby/DNS knowledge (and as i seen - it’s more about duck typing copy-pasted commands to shell) - and you ready to go!

I don’t want to retell everything that describe Octopress install guide and Jekyll internals. The basics of this blog engines can be described with simple words - everything is pre-generated and committed to Github pages as completely static site with all pages relinking and so on. Like a static cache instead of dynamic page generation (like Wordpress).

That’s why blogs on Jekyll-based engines is blazing-fast. And with Github protection you can be sure that your blog will be safe and alive in 99% of problems. I really believe in capabilities of Github team - dat guys know how to cook everything tasty.

Where is no any problem with customization and custom themes - just type some commands from your choosen theme site (like i did with Slim theme) - and it will be installed properly. Wanna change it? Go and edit simple HTML templates.

Even comments exist for this Blogs! Thanks to Disqus.

But event better is blogging itself - markdown is perfect choice for this as for me and simple editing of pre-generated files is fun. Preview on your local host is available, so you can easily fix somethnig before post.

Yea, I was amazed how simple and easy Octopress is. So, wanna try something new? Really hackable, easy-to-use and absolutely programmer’s way to blog? Go and checkout Octopress now - it really worth a try.
